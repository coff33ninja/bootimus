# Windows ISO Download Guide

Bootimus can download Windows installation ISOs from the **Internet Archive** — clean, original Microsoft images sourced from the [English Windows ISO Collection](https://archive.org/details/english_windows_collection).

## How It Works

1. Open the admin panel at `http://your-server:8081`
2. Click **Get Images** in the Images tab
3. Switch to the **Windows** tab
4. Browse the available releases — each entry shows the ISO filename and an Internet Archive download link
5. Click **Download**

The ISO is downloaded in the background. Progress is shown in the same panel.

## Source

All ISOs are hosted on the [Internet Archive](https://archive.org/details/english_windows_collection) — a free, trusted digital library. The collection contains original, unmodified Microsoft Windows images in English (EN-US).

After download, you can extract boot.wim and PXE-boot Windows just like any other image (Vista and newer).
